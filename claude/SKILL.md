# 中国 A 股/ETF/基金 实时价格查询

## 输入处理

用户调用方式：`/cn-stock-query [代码1] [代码2] ...`

从 `$ARGUMENTS` 中提取所有 6 位数字代码。若参数为空，提示用户提供代码：

> 请提供 6 位标的代码，例如：`/cn-stock-query 600519 510300 110011`

若用户使用名称而非代码，尝试匹配：

| 名称         | 代码   | 市场  |
|-------------|--------|------|
| 黄金ETF      | 518880 | 沪市  |
| 红利低波ETF  | 512890 | 沪市  |
| 沪深300ETF   | 510300 | 沪市  |
| 卫星ETF      | 563230 | 沪市  |
| 创业板ETF    | 159915 | 深市  |
| 中证500ETF   | 510500 | 沪市  |

无法确定代码时，向用户确认后再查询。

## 市场识别与防碰撞

对每个 6 位代码执行：

```
首位数字 → 市场推断
6       → sh（沪市股票）
0 或 3  → sz（深市股票）—— 注意：0 开头可能是场外基金，需二次确认
5       → sh（沪市 ETF）
1       → sz（深市 ETF）
其他    → 场外基金（2/4/7/8/9 开头）

对 0/3 开头：先查新浪接口，若返回空则转场外基金接口
查询返回后：校验名称与用户预期是否一致；不符时尝试另一市场前缀
```

## 查询：股票 / ETF

```bash
curl -s "https://hq.sinajs.cn/list={market}{code}" \
  -H "Referer: https://finance.sina.com.cn" \
  | iconv -f GBK -t UTF-8
```

**必须通过 `iconv -f GBK -t UTF-8` 转码，否则中文名称乱码。**

批量查询（逗号拼接）：

```bash
curl -s "https://hq.sinajs.cn/list=sh601991,sh518880,sz000001" \
  -H "Referer: https://finance.sina.com.cn" \
  | iconv -f GBK -t UTF-8
```

响应字段（引号内逗号分隔，0-based）：

| 索引 | 含义   |
|-----|--------|
| [0] | 名称   |
| [1] | 今开   |
| [2] | 昨收   |
| [3] | 最新价 |
| [4] | 最高   |
| [5] | 最低   |
| [8] | 成交量 |
| [9] | 成交额 |
| [30]| 日期   |
| [31]| 时间   |

涨跌额 = [3] − [2]，涨跌幅 = ([3] − [2]) / [2] × 100%

## 查询：场外基金

**优先（盘中估值）：**

```bash
curl -s "http://fundgz.1234567.com.cn/js/{code}.js"
```

> 此接口为 HTTP（无 TLS），请求仅含基金代码，不涉及用户敏感信息。

响应为 JSONP 格式，关键字段：
- `name`：基金名称
- `jzrq`：净值日期（最近确认）
- `dwjz`：单位净值（已确认）
- `gsz`：实时估算净值
- `gszzl`：估算涨跌幅(%)
- `gztime`：估值时间

**备用（确认净值）：**

```bash
curl -s "https://api.fund.eastmoney.com/f10/lsjz?fundCode={code}&pageIndex=1&pageSize=1" \
  -H "Referer: https://fund.eastmoney.com"
```

优先接口返回 `jsonpgz()` 空值时使用备用接口，标注 `（净值，非估值）`。

**QDII 延迟说明：**
名称含"QDII""纳斯达克""标普""海外""美国""全球"且净值日期落后 >2 天时，为正常现象，输出末尾附加：
`⏳ QDII基金，净值公布有 T+2~T+7 延迟`

## 输出格式

Markdown 表格：

```
| 代码   | 名称            | 类型  | 最新价  | 涨跌幅          | 更新时间            |
|--------|----------------|-------|--------|----------------|-------------------|
| 601991 | 大唐发电        | stock | 4.310  | 🔴 +1.41%      | 2026-03-18 15:00  |
| 518880 | 黄金ETF华安     | etf   | 10.606 | 🟢 -0.29%      | 2026-03-18 15:00  |
| 014978 | 华安纳指100联接C | fund  | 6.8897 | 🟢 -1.43%（估） | 2026-03-18 15:00  |
```

涨跌标识（A 股红涨绿跌）：
- 上涨：🔴 `+X.XX%`
- 下跌：🟢 `-X.XX%`
- 平盘：⚪ `0.00%`

其他规则：
- 价格保留原始小数位
- 场外基金盘中显示估算净值，涨跌幅后标注 `（估）`
- 非交易时段在表格下方注明：`⏸ 非交易时段，显示上一交易日收盘数据`

## 错误处理

| 场景              | 处理方式                                        |
|------------------|------------------------------------------------|
| 非 6 位代码       | 提示正确格式                                    |
| 接口返回空值       | 提示 "未找到标的 {code}，请确认代码是否正确"       |
| 接口超时 (>5s)    | 重试 1 次，仍失败提示 "数据源暂时不可用"           |
| 场外基金无估值     | 回退确认净值接口，标注 `（净值，非估值）`           |
| 市场前缀碰撞       | 校验名称，不符时尝试另一市场前缀                   |
| 中文乱码          | 检查是否遗漏 `iconv` 转码                        |

## 交易时间

- A 股/ETF：工作日 09:30-11:30, 13:00-15:00
- 场外基金估值：工作日 09:30-15:00
- QDII 基金净值：T+2～T+7 公布
