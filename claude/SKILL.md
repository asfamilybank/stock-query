# 全球股票/ETF/基金/指数 实时价格查询

## 输入处理

用户调用方式：`/stock-query [代码1] [代码2] ...`

从 `$ARGUMENTS` 中提取代码。若参数为空，提示用户：

> 请提供标的代码，例如：`/stock-query AAPL 00700 601991 510300`

支持的输入格式：
- **6 位纯数字** → A 股（股票/ETF/指数/基金），如 `601991`
- **5 位纯数字** → 港股，如 `00700`
- **英文字母** → 美股 ticker，如 `AAPL`、`TSLA`
- **`.` 开头** → 美股指数，如 `.DJI`、`.IXIC`、`.SPX`
- **中文/英文名称** → 尝试匹配常见标的（见下表）

常见标的速查：

| 名称         | 代码   | 市场     |
|-------------|--------|---------|
| 黄金ETF      | 518880 | A股沪市  |
| 红利低波ETF  | 512890 | A股沪市  |
| 沪深300ETF   | 510300 | A股沪市  |
| 上证指数     | 000001 | A股指数  |
| 深证成指     | 399001 | A股指数  |
| 创业板指     | 399006 | A股指数  |
| 腾讯         | 00700  | 港股     |
| 阿里巴巴(港) | 09988  | 港股     |

| 苹果         | AAPL   | 美股     |

| 英伟达       | NVDA   | 美股     |

| 道琼斯       | .DJI   | 美股指数  |

| 标普500      | .SPX   | 美股指数  |

无法确定代码时，向用户确认后再查询。

## 市场识别

```
if 5位纯数字           → hk{code}     港股
elif 以 . 开头          → us{code}     美股指数
elif 含英文字母         → us{CODE}     美股（ticker 大写）
elif 6位纯数字:
    000xxx → sh（沪市指数，注意与深市股票歧义）
    399xxx → sz（深市指数）
    6      → sh（沪市股票）
    0 或 3 → sz（深市股票，0 开头可能是场外基金需二次确认）
    5      → sh（沪市 ETF）
    1      → sz（深市 ETF）
    其他   → 场外基金

A股歧义处理：
- 000 开头：先查 sh{code}，名称含"指数"→指数；否则查 sz{code}→深市股票
- 000001 歧义：sh000001=上证指数, sz000001=平安银行
- 0/3 开头非指数：先查腾讯 sz{code}，无数据→转场外基金
- 查询后校验名称，不符时尝试另一市场前缀
```

## 查询：股票 / ETF / 指数（全球）

### 首选：腾讯财经（支持 A股/港股/美股）

```bash
# A 股
curl -s "https://qt.gtimg.cn/q=sh601991,sz000001" | iconv -f GBK -t UTF-8

# 港股
curl -s "https://qt.gtimg.cn/q=hk00700,hk09988" | iconv -f GBK -t UTF-8

# 美股 + 美股指数
curl -s "https://qt.gtimg.cn/q=usAAPL,usTSLA,us.DJI" | iconv -f GBK -t UTF-8

# 跨市场混合批量
curl -s "https://qt.gtimg.cn/q=sh601991,hk00700,usAAPL" | iconv -f GBK -t UTF-8
```

**必须通过 `iconv -f GBK -t UTF-8` 转码，否则中文名称乱码。**

API 前缀：A股 `sh`/`sz` + 6位 | 港股 `hk` + 5位 | 美股 `us` + ticker | 美股指数 `us` + `.XXX`

响应字段（引号内 `~` 分隔，0-based，各市场通用）：

| 索引 | 含义                                       |
|-----|---------------------------------------------|
| [0] | 市场标识（1=A股沪市, 51=A股深市, 100=港股, 200=美股）|
| [1] | 名称（中文）                                 |
| [2] | 代码                                        |
| [3] | 最新价                                      |
| [4] | 昨收                                        |
| [5] | 今开                                        |
| [6] | 成交量                                      |
| [30]| 日期时间                                     |
| [31]| 涨跌额（已算好）                              |
| [32]| 涨跌幅 %（已算好，直接使用）                    |
| [33]| 最高                                        |
| [34]| 最低                                        |

涨跌额和涨跌幅由接口直接返回，无需手动计算。

**频率限制：** 公共 API，过度调用会封 IP。健康频率：单次最多 100 个代码，间隔 ≥100ms。批量查询优先合并为单次请求。

### 备用：新浪财经（仅 A 股）

腾讯接口超时(>5s)或返回空时，A 股回退到新浪：

```bash
curl -s "https://hq.sinajs.cn/list={market}{code}" \
  -H "Referer: https://finance.sina.com.cn" \
  | iconv -f GBK -t UTF-8
```

新浪字段（逗号分隔）：[0]名称 [1]今开 [2]昨收 [3]最新价 [4]最高 [5]最低 [8]成交量 [30]日期 [31]时间

涨跌额 = [3]−[2]，涨跌幅 = ([3]−[2])/[2] × 100%

## 查询：场外基金（仅 A 股）

**优先（盘中估值）：**

```bash
curl -s "http://fundgz.1234567.com.cn/js/{code}.js"
```

> 此接口为 HTTP（无 TLS），请求仅含基金代码，不涉及敏感信息。

关键字段：`name` 名称, `dwjz` 单位净值, `gsz` 估算净值, `gszzl` 估算涨跌幅%, `gztime` 估值时间, `jzrq` 净值日期

**备用（确认净值）：**

```bash
curl -s "https://api.fund.eastmoney.com/f10/lsjz?fundCode={code}&pageIndex=1&pageSize=1" \
  -H "Referer: https://fund.eastmoney.com"
```

返回 `jsonpgz()` 空值时用备用接口，标注 `（净值，非估值）`。

**QDII 延迟：** 名称含"QDII""纳斯达克""标普""海外"等且净值落后 >2 天，附加：`⏳ QDII基金，净值公布有 T+2~T+7 延迟`

## 输出格式

Markdown 表格：

```
| 代码    | 名称      | 市场  | 最新价    | 涨跌幅          | 币种 | 更新时间            |
|---------|----------|-------|----------|----------------|------|-------------------|
| 601991  | 大唐发电  | A股   | 4.21     | 🔴 +4.47%      | CNY  | 2026-03-24 15:00  |
| 00700   | 腾讯控股  | 港股  | 511.000  | 🔴 +2.53%      | HKD  | 2026-03-24 16:00  |
| AAPL    | 苹果     | 美股  | 251.49   | 🟩 +1.41%      | USD  | 2026-03-23 16:00  |
| .DJI    | 道琼斯   | 美股  | 46208.47 | 🟩 +1.38%      | USD  | 2026-03-23 17:11  |
```

涨跌标识：
- **A 股/港股（红涨绿跌）**：🔴 `+X.XX%` / 🟢 `-X.XX%`
- **美股（绿涨红跌）**：🟩 `+X.XX%` / 🟥 `-X.XX%`
- **平盘**：⚪ `0.00%`

其他规则：
- 价格保留原始小数位
- 场外基金盘中标注 `（估）`
- 非交易时段注明：`⏸ 非交易时段，显示上一交易日收盘数据`

## 数据源降级策略

- A 股：腾讯财经 → 新浪财经 → 报错
- 港股/美股：腾讯财经 → 报错
- 场外基金：天天基金估值 → 东方财富净值 → 报错

## 错误处理

| 场景              | 处理方式                                        |
|------------------|------------------------------------------------|
| 代码无法识别       | 提示正确格式                                    |
| 腾讯超时/空值      | A股回退新浪；港股/美股提示不可用                  |
| 场外基金无估值     | 回退确认净值接口，标注 `（净值，非估值）`           |
| 市场前缀碰撞       | 校验名称，不符时尝试另一市场前缀                   |
| 中文乱码          | 检查 `iconv` 转码（腾讯和新浪均为 GBK）           |
